## 概要

- [J-Quants API](https://jpx.gitbook.io/j-quants-ja/api-reference) V2 を使用して、株の分析を行う
- 配当を出すプライム/スタンダード銘柄について、**権利落ち（指定日）後10営業日以内の最安値が指定日終値の95%未満**になった銘柄を抽出する
- データ取得・判定をすべて J-Quants API V2 内で完結する（外部サイトのスクレイピングなし）

## 用意するもの

- Python 3（**標準ライブラリのみ**。第三者パッケージのインストール不要）
- J-Quants APIの**APIキー**（[J-Quants Dashboard](https://jpx-jquants.com/dashboard/menu/?lang=ja)で発行）
  - 発行後、環境変数に設定する: `export JQUANTS_API_KEY=<発行したキー>`
  - APIキーに有効期限はない

## 使い方

```sh
export JQUANTS_API_KEY=<発行したキー>
python3 screen_dividend_drop.py [--month YYYY-MM] [--threshold 0.95] [--window 10] [--limit N] [--max-rps 3.0] [--csv 出力先.csv]
```

| オプション | 既定 | 説明 |
|---|---|---|
| `--month` | 約4ヶ月前 | 対象の権利確定月（YYYY-MM）。未指定時はデータ提供範囲内で安全な直近月を概算採用し警告を表示 |
| `--threshold` | `0.95` | 指定日終値に対する比較割合 |
| `--window` | `10` | 指定日以降に最安値を探す営業日数 |
| `--limit` | なし(全件) | 候補barsの処理数を制限（開示日スキャンのコール数は減らない）。規模が大きい月での動作確認用 |
| `--max-rps` | `3.0` | API リクエストの上限レート（秒あたり）。レート制限対策のスロットル |
| `--csv` | なし | 該当銘柄（コード/銘柄名/市場/指定日）をCSV出力（Excel向けにBOM付きUTF-8） |

- 該当銘柄は標準出力へ、進捗・情報・エラーは標準エラー出力へ出る（`... | grep`等で該当行のみ扱える）。

## 仕組み

1. `/fins/summary`（決算サマリ）を開示日（平日）で反復取得し、配当を出すFY（期末 `DivFY`）・2Q（中間 `Div2Q`）から権利確定日(`CurPerEn`)を導出
2. 指定日 = 権利確定日の **2営業日前**（営業日は株価系列から判定。別途のカレンダーAPI・祝日ライブラリは不要）
3. `/equities/master` で市場区分(`MktNm`)・銘柄名(`CoName`)を付与（プライム/スタンダードのみ）
4. `/equities/bars/daily` で指定日終値と以降の営業日の最安値を比較し、`min < threshold × 指定日終値` を抽出

## 注意事項

- **データ提供範囲**: 取得できる日付範囲は契約プランに依存する（ローリング。例: 約2年・直近約12週は未提供）。`--month` がこの範囲外だと結果が空、またはAPIの範囲外メッセージが `[error]` で表示される。
- **レート制限**: 下位プラン（無料等）はレート制限が厳しい。人気月（3月・9月決算が集中）は候補が多くAPIコールが大量になるため、`--limit` で件数を絞り、`--max-rps` を下げて実行すること。本ツールは 429 received 時に指数バックオフで再試行するが、プランのクォータ枠の回復には時間がかかる場合がある。なお、`--limit` を指定しても開示日スキャン（約67コール/月）は減らないことに注意。
- **対象ユニバース**: 「配当を出す銘柄」が対象。優待のみ・無配当の銘柄は対象外（J-Quantsの配当エンドポイント `/fins/dividend` はプランによっては利用不可のため、`/fins/summary` から導出している）。
- **権利確定日の近似**: 権利確定日（基準日）は決算サマリの会計期末日 `CurPerEn` で近似している。基準日が期末と異なる稀な銘柄（中間配当の実際の基準日が 9/30 以外など）では指定日の算出がズレる場合がある。
- **終値**: 調整前の生値（V2の `C`）で比較する。株式分割・併合をまたぐ期間では調整後終値（`AdjC`）の利用も検討すること。
